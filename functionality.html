Start up screen

When the app starts, the user is presented with the focus starting in a place where he or she can type her name and place the NEXT button to continue. Entering name is not required, but it is recommended for anyone wishing to play this game.

The user's name is displayed in this screen and that's where the fun begins. The user is asked to guess "How many children are in Rawi's Family?" How so! Well, there is not RIGHT or WRONG answers to this question. The end-user will make their best guess by entering an amount for boys and girls. The error message "Please enter a valid numbers will be displayed" if users failed to enter number of girls or boys in Rawi's household. The answer will be displayed after the "SEND" button is clicked.
